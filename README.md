# Supermarket-Billing-System-

The SmartBuy Supermarket Billing System is a Python-based application designed to automate customer billing processes. The system allows cashiers to enter product details, calculate bills, apply discounts automatically, and generate customer receipts efficiently.

# Features

    Multiple product entry
    Automatic bill calculation
    10% discount for bills above Le 500
    Receipt generation
    Multiple customer processing
    User-friendly interface

# How the Program Works

    The cashier enters the number of products purchased.
    Product details (name, quantity, and price) are entered.
    The system calculates the total cost of each product.
    The overall bill is calculated automatically.
    A 10% discount is applied if the total bill exceeds Le 500.
    The final amount payable is displayed.
    A receipt is generated for the customer.
    The system can process multiple customers.

# Requirements

    Python 3.x

# How to Run the Program

    Save the Python code as billing_system.py.  
    open a terminal or command prompt.
    Navigate to the project folder.
    Run the program using:

python billing_system.py

# Problem Solved

This system eliminates manual billing errors, reduces waiting time at checkout counters, improves accuracy, and enhances customer satisfaction.
